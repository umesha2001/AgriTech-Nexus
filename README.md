# AgriTech App

A comprehensive mobile application designed to connect agricultural professionals, farmers, experts, and community leaders in a unified platform for sharing knowledge, facilitating communication, and promoting sustainable farming practices.

## Overview

AgriTech is a Flutter-based mobile application that provides a seamless communication and community platform for the agricultural sector. The app enables users to connect with government officers, agricultural experts, community leaders, farmers, buyers, and sellers through communities, voice/video calls, and real-time messaging.

## Features

### 1. **Authentication**
- User registration with role-based selection (Government Officers, Agricultural Experts, Community Leaders, Sellers, Buyers, Farmers)
- Secure login system
- Password recovery functionality

### 2. **Communities**
- Join and explore different agricultural communities
- Connect with peers in your sector
- Share knowledge and best practices
- Create new communities for specific agricultural focuses

### 3. **Voice & Video Calls**
- One-to-one voice calls
- Video calling capabilities
- Group calling support (add participants)
- Call history with timestamps
- Mute/unmute and speaker controls
- Real-time call duration display

### 4. **Messaging**
- Real-time text messaging
- Message history
- Chat rooms for different communities
- Unread message indicators
- Media sharing capabilities

### 5. **User Profiles**
- Customizable user profiles
- Profile management
- User role display
- Quick access to main features from profile

### 6. **Dashboard Navigation**
- Intuitive bottom navigation bar
- Easy switching between Communities, Calls, and Chats
- User profile access
